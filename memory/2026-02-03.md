```bash
sleep 1 && \
osascript -e 'tell application "System Events" to keystroke "l" using {command down}' && \

osascript -e 'tell application "System Events" to keystroke "YOUR PROMPT HERE"' && \
sleep 0.3 && \
osascript -e 'tell application "System Events" to keystroke return'
```

1. **Peekaboo** - Focuses correct Windsurf window by workspace name
2. **osascript** - Activates Windsurf and sends keystrokes
3. **Cmd+L** - Opens Cascade chat
4. **Direct typing** - Types prompt (clipboard paste doesn't work in Cascade)
5. **Enter** - Submits prompt
